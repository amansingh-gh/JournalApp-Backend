# 📓 JournalApp Backend

A production-ready journaling backend built with **Spring Boot**, **Spring Security**, and **JWT Authentication**. The project demonstrates secure user authentication, RESTful API development, CRUD operations, and scalable backend architecture following enterprise Java development practices.

---

## ✨ Features

- 🔐 JWT Authentication & Authorization
- 👤 Secure User Registration & Login
- 📖 Personal Journal CRUD Operations
- 🛡️ Spring Security Integration
- ✅ Input Validation
- ⚡ RESTful API Architecture
- 📂 Layered MVC Design
- 🗄️ Database Persistence
- 🚀 Frontend Ready APIs

---

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Security
- JWT
- Hibernate / JPA
- Maven
- MySQL
- REST APIs
